APENAS O MELHOR DO MUNDO







-- who is Neymar? --



Para o cego, é luz.
Para o faminto, é o pão.
Para o sedento, é a fonte de água.
Para o morto, é a vida.
Para o enfermo, é a cura.
Para o prisioneiro, é a liberdade.
Para mim, é tudo.
O Neymar acha, eu concordo. O Neymar fala, eu escuto. O Neymar erra, eu passo pano. Se o Neymar tem 1 fã, esse sou eu.
